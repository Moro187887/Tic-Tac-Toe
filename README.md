Tic Tac Toe - Python Command Line Game

This is a simple Tic Tac Toe game for two players.
It runs in the command line with Python.

How to start the game:

1. Open the outputs folder.
2. Double-click start_tic_tac_toe.bat.

You can also start it from PowerShell with this command:

python "C:\Users\Kraem\Documents\Codex\2026-06-03\mach-mir-tic-tac-toe-mit\outputs\tic_tac_toe.py"

How to play:

The board has nine fields, numbered from 1 to 9.

 1 | 2 | 3
---+---+---
 4 | 5 | 6
---+---+---
 7 | 8 | 9

Player X starts.
After that, Player O takes a turn.
Each player chooses a free field by typing a number from 1 to 9.

The goal is to get three symbols in a row.
This can be horizontal, vertical, or diagonal.

You can quit the game by typing q.

If all fields are full and nobody has three in a row, the game ends in a draw.
